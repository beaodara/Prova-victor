//diferença ts e js // Em Angular, um endpoint é uma rota definida na API ou serviço que a aplicação se comunica para obter ou enviar dados. O HttpClient é usado para realizar solicitações HTTP aos endpoints. Os serviços encapsulam a lógica de comunicação com os endpoints e retornam os dados para os componentes. //organizar html //Para organizar o HTML considerando a estrutura descrita, podemos criar três componentes: AppHeader, AppContent e AppFooter. O mesmo se aplicaria aos arquivos HTML correspondentes (app-header.component.html, app-content.component.html e app-footer.component.html), onde cada componente poderia ser estilizado e estruturado individualmente.
